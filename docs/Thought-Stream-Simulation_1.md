# Unspool: Thought Stream Simulation

One simulated user: **Maya**, 27, grad student + part-time barista, ADHD diagnosed at 22. Lives alone, has a cat named Beans. Working on her thesis, frequently overwhelmed.

Legend:
- **UT** = User Thought (what Maya types)
- **UnsT** = Unspool Thought (what Unspool responds)
- **UM** = User Memory (what Maya remembers in her head — we can't see this, but it matters)
- **UnsM** = Unspool Memory (what Unspool derives/stores)
- **UA** = User Action (something Maya does in the real world or in other apps)
- **UnsA** = Unspool Action (something Unspool does — calendar, notification, internal processing)
